# SocketIO_server
## Ejecucion
- clonamos el repositorio `git clone https://github.com/02Alexis/SocketIO_server.git`
- entramos a la carpeta `cd SocketIO_server`
- ejecutamos el comando `npm install`
- luego ejecutamos `npm start`
